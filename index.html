03

Install dependencies

Install the three core libraries needed for training and serializing your model.

bash
pip install scikit-learn pandas joblib
04

Write the training script

Your train.py trains a sentiment classifier and saves the result as a .pkl file. Here is a minimal example:

python · train.py
from sklearn.pipeline import Pipeline
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.linear_model import LogisticRegression
import joblib

# Training data
texts  = ["I love this", "Great product", "Terrible", "Very bad"]
labels = [1, 1, 0, 0]

# Build and train pipeline
model = Pipeline([
    ("tfidf", TfidfVectorizer()),
    ("clf",  LogisticRegression()),
])
model.fit(texts, labels)

# Save the trained model
joblib.dump(model, "sentiment_model.pkl")
print("✓ Model saved → sentiment_model.pkl")
05

Run training

bash
python train.py
expected output ✓ Model saved → sentiment_model.pkl
Phase 02 Build the REST API
FastAPI uvicorn
06

Install FastAPI and Uvicorn

FastAPI is a modern, high-performance web framework. Uvicorn is the ASGI server that runs it.

bash
pip install fastapi uvicorn
07

Create the API file

bash · windows
type nul > main.py
08

Write the API

This loads the saved model and exposes two endpoints: a health check and a prediction route.

python · main.py
from fastapi import FastAPI
import joblib

app = FastAPI(title="Sentiment API", version="1.0")

# Load the model once at startup
model = joblib.load("sentiment_model.pkl")

@app.get("/")
def health_check():
    return {"status": "ok", "message": "AI Model is running 🚀"}

@app.post("/predict")
def predict(text: str):
    label = model.predict([text])[0]
    return {
        "input":      text,
        "prediction": int(label),
        "sentiment":  "positive" if label == 1 else "negative"
    }
Phase 03 Run & Test
09

Start the server

Optional · Phase 04
Phase 04 Containerize with Docker

Use Docker when you need to ship the project to a server, share it with a team, or ensure it runs identically on any machine — regardless of local Python installations.

12

Create the Dockerfile

bash · windows
type nul > Dockerfile
13

Write the Dockerfile

dockerfile
FROM python:3.10-slim

WORKDIR /app

COPY . .

RUN pip install --no-cache-dir \
    scikit-learn pandas joblib fastapi uvicorn

EXPOSE 8000

CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
14

Build and run

bash
# Build the image
docker build -t ai-model .

# Run the container
docker run -p 8000:8000 ai-model

The API is now accessible at http://localhost:8000 — running inside an isolated container.
